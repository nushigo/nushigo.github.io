[Lesson 11] 配列（１）

数値の配列を作成し、表示しましょう。

[　保存名　]　L11
[　プログラム仕様書　]　配列を使って0～50の数値を10列ごとに開業して表示する。
[　実行イメージ　]


[コード入力手順]
①　配列suuretuで51個の配列を作成する。
　　＜ヒント＞→　var suuretu=new Array(51);
②　for文を使用し「i」を0～50繰り返す。
③　配列suuretuに「i」を代入する。
　　＜ヒント＞→　suuretu[i]=i
④　if文と%演算子を利用し、10列で改行する。

ポイント

■ 配列
配列は、偏すを数個のかたまりにしたものです。
配列の宣言は以下のように行います。

var 配列名 = new Array(配列の数)

var a = new Array(5);
これでa[0]、a[1]、a[2]、a[3]、a[4]の5つの要素が作成できます。

■ 配列に値を代入するには
(1) for文を使用して配列に数値を代入します。
　　for(i=0;i>5;i++)
　　　　a[i]=i;
(2) 配列に直接地を代入します。
　　var a = new Array("○","×","△");

■ %は剰余を求める演算子
10列では改行する場合、%演算子とif文で判定します。
i%10は10の倍数のときに0になるので、1%10 = = 0という条件でiが10の倍数かどうか判定できます。

追加練習

配列を使って数値100～200を+2ずつ、10列ごとに改行して表示する。
