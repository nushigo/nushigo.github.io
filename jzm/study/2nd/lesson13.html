[Lesson 13] 配列（３）

配列を使用し、テストの平均点を算出するプログラムを作成しましょう。

[　保存名　]　L13
[　プログラム仕様書　]　平均点を求める関数を作成する。
配列で5回のテストの点数を作成し、関数に配列のデータを返して平均点を計算する。
[　実行イメージ　]


[コード入力手順]
①　heikin(d,n)関数を作成する。
function heikin(d,n){
　　if(n<=0){
　　　　total=0;
　　　　for(i=0;i<n;i++) total += d[i];
　　　　return total/n;
　　}
｝

②　配列tensuで5つの数値の配列を作成し、点数を代入する。
③　変数「n」に5を代入する。 ④　heikin()関数を使用して、配列の数値から平均を出す。
　　　　＜ヒント＞→　heikin(tensu,n)
⑤　配列の数値を表示させる。
　　　　＜ヒント＞→　joinメソッド

ポイント

■ 平均を求めるアルゴリズム
平均を求めるには、合計を出し、数回で割ります。
点数と回数を引数に使うので、heikin(d,n)とします。
電卓で集計するときはAC(All Clear)キーを押して0にしてから始めるのと同じように初期値を必ず0にしておきます。
total=0;
そして、集計するデータを「total」に繰り返し加算していきます。今回は点数を配列で受け取ります。
total += d[i];（または、total=total+[d]）
合計を回数で割って、平均を計算しreturn文で結果を返します。
return total / n;

■ joinメソッド
document.write(配列名.join("区切りの文字");
「join」メソッドは、配列の要素を区切り文字で連結します。
「join("区切り文字")」を指定すると、要素が区切り文字つきで表示されます。
()内の指定を省略すると、各要素は「,」で区切られます。

■ 代入演算子
代入演算子は、ある値を変数に代入するために用います。
基本的には、代入演算子の右辺に記述された値や式を左辺に記述した変数に代入します。
例 内　容
a = b bをaに代入する
a += b aにa+bを代入する
a -= b aにa-bを代入する
a *= b aにa*bを代入する
a /= b aにa/bを代入する
a %= b aにa/bの剰余を代入する
