アドバイス

オブジェクト（配列）の数を取得する
プロパティやメソッドの中には、複数のオブジェクトで使用できるものがあります。
lengthプロパティはHTMLファイル内のオブジェクトや配列の数を取得します。
・サポートしているオブジェクトの一例
Button Checkbox Form
Frame Image Radio
Text Textarea Window
・サポートしている配列の一例
elements forms frames
images links options

＜例＞
var moji="あいうえお";

　　document.write("フォームの数",document.forms.length,"<br />");
