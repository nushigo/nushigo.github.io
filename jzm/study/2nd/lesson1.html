[Lesson 1] for 文（１）

for文を使用し、文字列を繰り返し表示しましょう。

[　保存名　]　L01
[　プログラム仕様書　]　for文を使用し、文字列「♪　にっぽん　(^o^)/~」を3回繰り返して表示する。
[　実行イメージ　]


[コード入力手順]
①　テキストエディタ（メモ帳など）を起動し、以下のコードを入力する。

前略前後
<script>
	for(i=0;   ;i++){
		              ("<h2>♪　にっぽん　(^o^)/˜</h2>");
	}
</script>

「L01.html」で保存し、ブラウザで確認する。

ポイント

■ for文の書き方基本形

for(初期値, 終了条件, 増減式){
	繰り返す処理・・・;
}

例：　for(i=0;i<3;i++)　　iは0から3未満まで1ずつ増加する。
例：　for(i=5;i>0;i--)　　iは5から0より大きい間1ずつ減少する。
例：　for(i=0;i<30;i=i+5)　　iは0から30未満まで5ずつ増加する。

■ {}の省略
繰り返す処理のコードが1行の場合は{}を省略できます。

追加練習

2行目に「チャチャチャ」の文字を追加し、繰り返しを5回に変更する。

実行イメージ

文字列「チャチャチャ」の前にはスペース（空白）を入力しています。


答え

分からなかった場合などは答えを見てみましょう。


<script>
	for(i=0;i<3;i++){
		document.write("<h2>♪　にっぽん　(^o^)/˜</h2>");
	}
</script>




<script>
	for(i=0;i<5;i++){
		document.write("<h2>♪　にっぽん　(^o^)/˜</h2>");
		document.write("<strong>　　　　　　チャチャチャ</strong>");
		全角スペースが6つ入ります。
	}
</script>
