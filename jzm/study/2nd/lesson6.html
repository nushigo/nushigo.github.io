[Lesson 6] if～else 文（１）

if文の条件分岐と%演算子を使用し、文字列を表示しましょう。

[　保存名　]　L06
[　プログラム仕様書　]　文字列「こんにちは」と「ニーハオ」を交互に6回繰り返して表示する。
交互の判定はif文と%演算子を使用する。
[　実行イメージ　]


[コード入力手順]
①　for文を使用し、「i」を6回繰り返し、改行を行う。
②　「i」を2で割った余りが「0」の時には「こんにちは」を表示する。
　　　　＜ヒント＞→　if(i%2==0)
　　　　　　　　　　　　document.write("こんにちは");
③　それ以外は、「ニーハオ」と表示する。

アドバイス

剰余
剰余（%）とifを使用すると条件判定の繰り返しが容易に行えます。
例えば、「i」が0～10の数値の繰り返しの場合は、
「i%2」は「0」、「1」の繰り返しになり、「0」の時は…「1」の時は…と条件分岐が行えます。
「i%3」は「0」、「１」、「２」の繰り返しになります。

ポイント

■ if文
if文は複数処理に条件をつけたいときに使用します。
人間の行動だと、例えば500円のものをフリーマーケットで発売するとします。
その場合「500円以上なら、お釣りを返す」、「500円なら商品を渡す」、「500円以下はお断り」の様に対応していくと思います。
そのように条件毎に違う処理をコンピューターにやってもらうために指示するためのコードがif文なのです。

■ else文
else文はif文で指示した条件の対象にならなかった際に行う処理を記載するために使用します。
if文の説明で例として挙げた「人間の行動」で例えると、「500円以下はお断り」の対応にあたります。
if文は条件をつけた処理、else文は対象外の処理と覚えると分かりやすいでしょう。

■ if文の書き方基本形
if(条件1){
　　条件1が真のときに行う処理;
}
else if(条件2){
　　条件2が真のときに行う処理;
}
else {
　　条件1、条件2に合わない場合の処理;
}


■ 比較演算子
例 内　容
a==b aとbが等しい
a!=b aとbが等しくない
a>b aがbより大きい
a<b aがbより小さい
a<=b aがb以上
a>=b aがb以下

追加練習

文字列「こんにちは」と「ニーハオ」と「アンニョンハセヨ」を交互に6行表示する。
