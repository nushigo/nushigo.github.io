[Lesson 7] if～else 文（２）

if文の条件式を使用し、文字列を表示しましょう。

[　保存名　]　L07
[　プログラム仕様書　]　文字列「●」、「◇」、「▲」を1行じに表示することを4回繰り返す。
ただし文字色を「●」は青、「◇」はライム、「▲」は赤にする。
なお、1行目と3行目のフォントサイズは3、2行目と4行目のフォントサイズを7にする。
[　実行イメージ　]


[コード入力手順]
①　for文を使用し、「i」を4回繰り返し、改行を行う。
②　if文を使用し、「i」が1と3のときは、フォントサイズを3にする。それ以外はフォントサイズを7にする。
　　　　＜ヒント＞→　if(i==1 || i==3)
　　　　　　　　　　　　document.write("<font size=3>");
③　カウンタ変数「j」を使用し、文字表示を3回繰り返し、「j」が1のときはフォントサイズの色を青にし、「●」を表示する。
④　「j」が2のときはフォントの色をライムにし、「◇」を表示する。
⑤　「j」が3のときはフォントの色を赤し、「▲」を表示する。

ポイント

■ 論理演算子
複数の条件を組み合わせる時に論理演算しを使用します。
例 内　容
条件式a && 条件式b aq AND b（かつ）
条件式a || 条件式b a OR b（または）
!条件式a NOT a（否定）

■ 色の指定
JavaScriptでは色名、RGB16進数、RGB25進数の3種類の方法で色の指定ができます。
詳しくは巻末の「補足」を参照してください。
色　名 RGB16進数
#RRGGBB		 RGB256進数
(R,G,B)
blue #0000FF (0,0,255)
lime #00FF00 (0,255,0)
red #FF0000 (255,0,0)

答え

分からなかった場合などは答えを見てみましょう。


<script>
for(var i=1;i<5;i++){
	if(i==1){
		document.write("<font size=3 color='blue'>");
	}
	else if(i==2){
		document.write("<font size=7 color='lime'>");
	}
	else if(i==3){
		document.write("<font size=3 color='red'>");
	}
	else{
		document.write("<font size=7>");
	}
	document.write("●◇▲<br />");
	document.write("</font>")
}
</script>
