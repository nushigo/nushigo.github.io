[Lesson 9] 関数（２）

関数を使用し、空白文字列を表示しましょう。

[　保存名　]　L09
[　プログラム仕様書　]　空白文字列（スペース）を生成する関数を作る。
[　実行イメージ　]


[コード入力手順]
①　空白文字列を作成するspc(n)関数を作成する。
　　・sの値をヌル文字列（なにもない文字列）に初期化する。
　　・for分を使用してn回の繰り返しを行い空白文字を足していく。
for(i=1;i<n;i++){
　　s=s+" ";
　　return s;
}

②　イメージファイル「a.gif」を表示する。
③　for文の使用して「i」が1→3→5→7→9になるように繰り返し行う。
④　関数spc(i)でi個分の空白文字列と作成し、「スペースi」とイメージファイルの間に表示させる。

ポイント

■ <pre>タグ
空白文字列が表示されないときは、<pre>タグを使います。
<pre>タグはコードを記述した空白や改行をそのまま表示することができます。
