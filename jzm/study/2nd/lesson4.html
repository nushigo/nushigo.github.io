[Lesson 4] for 文（４）

for文を使用した多重ループでイメージファイルを表示しましょう。

[　保存名　]　L04
[　プログラム仕様書　]　イメージファイル「001.gif」、「002.gif」、「003.gif」を1行に4回ずつ繰り返して表示する。
[　実行イメージ　]


[コード入力手順]
①　イメージファイル「001.gif」、「002.gif」、「003.gif」を用意する。
②　for文を使用し、「i」を3回繰り返し、改行を行う。
③　for文を使用し、「j」を4回繰り返し、画像を表示する。
④　カウンタ変数「i」を使用し、イメージファイルを表示する。
＜ヒント＞→　「"00"+i+".gif」
⑤　イメージファイルの大きさは各自整えて表示させる。
＜ヒント＞→　width, height

ポイント

■ 多重ループ
ループの中にループが入ることを多重ループといいます。多重ループでは内側のループから実行されます。
iの値 jの値 イメージファイル

改行

答え

分からなかった場合などは答えを見てみましょう。


<script>
for(var i=1;i<4;i++){
	for(var j=1;j<5;j++){
		document.write("<img src='../../common/img/L04-nm"+i+".png' width='150px'
		 height='150px'>");
	}
	document.write("<br />");
}
</script>
